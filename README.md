### Mouse / Trackpad Spotlight

- マウス・トラックパッド・タッチ操作で動く「スポットライト」演出のサンプル
- カーソル位置に応じて、画面を暗く覆うラジアルグラデーションを表示
- CSSカスタムプロパティ（`--x`, `--y`, `--size`, `--soft` など）で見た目を制御
- `pointermove` を使用し、マウス・トラックパッドの両方に対応
- タッチデバイスでも指の位置に追従
- キーボード操作対応  
  - 矢印キー / WASD：スポットライト移動  
  - Homeキー：中央に戻す
- レスポンシブ対応（モバイルではスポットライトを大きく表示）
- JavaScript・CSS・HTMLのみで動作（外部ライブラリ不要）
- アクセシビリティ配慮  
  - `tabindex` によりキーボードフォーカス可能  
  - `aria-label` を設定

#### 改善可能な点

##### パフォーマンス
- 衝突判定を毎フレーム全要素で行っている → 空間分割で最適化
- 敵の移動を毎フレーム計算 → 必要時のみ更新
- `querySelectorAll`を毎回実行 → 配列にキャッシュ

##### ゲームプレイ
- 難易度曲線が急すぎる → レベル間の調整
- 敵の動きがランダムすぎる → プレイヤーを追跡する敵
- 体力回復システムがない → ハートアイテム追加
- スコアシステムが単純 → 時間ボーナス、コンボシステム

##### UI/UX
- ポーズ画面がない → ポーズメニュー追加
- レベル開始時の準備時間なし → カウントダウン追加
- ダメージ時のフィードバックが弱い → 画面フラッシュ、音強化
- 進行状況が見えない → プログレスバー

##### コード品質
- マジックナンバーが多い → 定数化
- 関数が長い → 分割
- エラーハンドリングなし → try-catch追加
- グローバル変数使用 → クラス化/モジュール化

##### 機能追加
- セーブ/ロード機能なし → localStorage使用
- サウンドのON/OFF切り替えなし → 設定メニュー
- リーダーボード/ハイスコアなし
- チュートリアルなし → 最初のレベルでガイド
- モバイル最適化不足 → タッチ操作改善

##### ビジュアル
- パーティクルエフェクトなし → 星収集時のエフェクト
- トレイルエフェクトなし → プレイヤーの軌跡
- 背景が静的 → アニメーション追加
- スポットライトが単調 → 明滅効果

##### バグ修正
- 壁に引っかかることがある → 衝突判定の改善
- 連続ダメージを受ける → 無敵時間追加
- 敵が壁を突き抜ける可能性 → 境界チェック強化
